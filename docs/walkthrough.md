# LPsaitou セットアップ手順書

## 完了した手順

1.  **リポジトリのクローン**: `https://github.com/Azamarusuisan/LPsaitou.git` からソースコードを取得しました。
2.  **プロジェクトの確認**: プロジェクトが静的ウェブサイト (HTML/CSS/JS) であることを確認しました。Node.jsの依存関係は不要なため、`npm install` はスキップしました。
3.  **アプリケーションの起動 (再起動)**: ユーザーの要望により、別のサーバー (`http-server`) を使用してポート8080で再起動しました。
4.  **モバイルレイアウトの改善**:
    -   モバイル表示時に画像が大きすぎる問題を修正するため、`styles-mobile.css` を編集し、画像幅を `120%` から `100%` に変更しました。
    -   **ボタン機能の実装**: モバイル版で非表示になっていたメディアリンクボタン（Google, Yahoo等）を有効化し、2列レイアウトで配置しました。
5.  **Gitへのプッシュ**: 修正内容をコミットし、リモートリポジトリへプッシュしました。

## 確認方法
アプリケーションはローカルで実行されています。以下のURLでアクセスしてください。
-   [http://localhost:8080](http://localhost:8080)

(詳細なポート番号はターミナルの出力を確認してください)
