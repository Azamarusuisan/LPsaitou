# アドサポ10 ランディングページ

広告運用代行サービス「アドサポ10」のランディングページです。

## 概要

このLPは、広告運用代行サービス「アドサポ10」のプロモーション用に作成されました。
月額10万円からの完全固定料金でプロの広告運用を提供するサービスを紹介しています。

## ファイル構成

```
LP作成カーソル/
├── index.html      # メインHTMLファイル
├── styles.css      # スタイルシート
├── script.js       # JavaScriptファイル
└── README.md       # このファイル
```

## セクション構成

1. **ヒーローセクション**
   - サービス名とメインメッセージ
   - 3つの主要なメリット
   - CTAボタン

2. **お悩みセクション**
   - 広告運用に関する5つのお悩みポイント

3. **解決策セクション**
   - サービス紹介
   - 対応媒体リスト（8つのプラットフォーム）

4. **対応業務セクション**
   - 6つのサービス内容

5. **CTAセクション**
   - キャンペーン情報と申し込みボタン

6. **お問い合わせフォーム**
   - 資料請求・お問い合わせフォーム

## 使い方

1. `index.html`をブラウザで開く
2. 各セクションを確認
3. お問い合わせフォームから送信（現在はコンソールに出力されます）

## カスタマイズ

### 色の変更

`styles.css`の`:root`セクションでカラーパレットを変更できます：

```css
:root {
    --primary-orange: #FF6B35;
    --primary-blue: #004E89;
    /* ... */
}
```

### フォーム送信の実装

`script.js`のフォーム送信処理を、実際のバックエンドAPIに接続してください。

現在はコンソールに出力するだけの実装になっています。
